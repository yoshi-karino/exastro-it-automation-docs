はじめに¶

本書では、ITAのメニュー作成機能を運用する為のシステム構成と環境構築について説明します。

メニュー作成機能を利用するにあたっては、ITAの基本機能が構築済であることが前提です。
ITA基本機能の構築に関しては、「システム構成／環境構築ガイド_基本編」をご覧ください。
